Commenting in html :-
In HTML comments are used to add notes or explanations within the code that are not visible on the web page they are useful for developers to provide information or .reminders about specific sections of code.
Comments in html written between <!--text --> and anything within tags will be ignored by the browser when rendering the web page. For Commenting in html select particular text with curser which we dont want to rendered on web page and just hit ctrl+/ and we get "<!--text-->
example: 
<!-- this text will not render on browser because we comment it out-->
it will show in source code.
